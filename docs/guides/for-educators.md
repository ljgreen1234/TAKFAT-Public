# TAKFAT-Yawn-Index for Educators

## Overview for Teachers & Homeschool Parents

The TAKFAT-Yawn-Index project offers a unique opportunity to teach scientific method, data collection, physiology, and ethics through an engaging, accessible citizen-science experiment.

Students will:

1. **Practice Scientific Method**

2. **Develop Data Literacy**
   - Organize information systematically

4. **Understand Research Ethics**

   - Data sharing responsibilities

5. **Build Practical Skills**

   - Attention and self-awareness
   - Documentation and record-keeping
   - Collaboration and data sharing

### Timeline Options

#### One-Week Unit
- Day 1: Introduction and hypothesis development
- Days 2-8: Data collection
- Days 9-10: Analysis and presentation

#### Two-Week Extended Unit
- Week 1: Background learning + data collection
- Week 2: Data collection + analysis + presentation

#### Ongoing Project
- Introduce as part of health/science curriculum
- Students track for one week each marking period
- Compare across seasons or school year

### Lesson Plan Template

**Day 1: Introduction (45-60 minutes)**

1. **Hook** (10 min): Watch video of contagious yawning, discuss why
2. **Background** (15 min): Teach about autonomic nervous system, yawning research
3. **Project Introduction** (15 min): Explain citizen science and the project
4. **Hypothesis Development** (10 min): Students predict their yawn patterns
5. **Setup** (10 min): Create anonymous IDs, prepare tracking sheets

**Days 2-8: Data Collection**

- Daily 5-minute check-in
- Students share interesting observations
- Troubleshoot tracking challenges
- Reinforce consistency

**Days 9-10: Analysis and Presentation**

1. **Data Organization** (30 min): Enter data into spreadsheets
2. **Visualization** (30 min): Create graphs and charts
3. **Analysis** (30 min): Identify patterns, test hypotheses
4. **Presentation** (45 min): Students share findings
5. **Reflection** (15 min): What did we learn about science?

## Classroom Activities üé®

### Activity 1: Yawn Contagion Experiment

**Time**: 20 minutes

**Procedure**:
1. Show video of people yawning
2. Count how many students yawn
3. Calculate percentage
4. Discuss mirror neurons and empathy

### Activity 2: Create a Class Yawn Graph

**Time**: 45 minutes

**Procedure**:
1. Each student contributes their daily totals
2. Create large wall chart
3. Calculate class averages
4. Identify patterns (day of week, time of day)

### Activity 3: Time-of-Day Analysis

**Time**: 30 minutes

**Procedure**:
1. Aggregate all yawn times from class
2. Create histogram by hour
3. Discuss circadian rhythms
4. Compare to predictions

### Activity 4: Context Comparison

**Time**: 30 minutes

**Procedure**:
1. Categorize yawns by activity
2. Compare yawn rates during different activities
3. Discuss potential reasons
4. Relate to attention and arousal

### Activity 5: Design Your Own Yawn Study

**Time**: 60 minutes (or homework)

**Procedure**:
1. Students propose new research questions
2. Design modified tracking protocols
3. Present to class for feedback
4. Discuss feasibility and ethics

## Privacy and Parental Consent üîí

### Required Steps

1. **Send Permission Letter**: Use template in `docs/consent-forms/` (to be created)
2. **Obtain Signed Consent**: Required for all minors
3. **Explain Anonymization**: Ensure parents understand privacy protections
4. **Provide Opt-Out**: Participation must be voluntary

### Sample Parent Letter (Brief)

```
Dear Parents/Guardians,

Our class is participating in a citizen-science project studying 
yawning patterns. Students will track when they yawn for one week. 
This teaches scientific method, data collection, and biology.

All data is ANONYMOUS - no names, addresses, or identifying 
information will be shared. Students create code names (like 
"YI-2010-JS-123"). Data is shared publicly but cannot be linked 
to individual students.

This is voluntary. Students who don't participate will do an 
alternative activity. Please sign below if you approve.

More information: [GitHub repository link]

___ I give permission for my child to participate
___ I do NOT give permission (alternative activity provided)

Parent Signature: _________________ Date: _______
```

### Alternative Activities

- Research report on yawning science

## Data Submission Options üìä

### Option 1: Individual Submissions

Each student submits their own anonymized data (with parental consent and teacher review).

### Option 2: Class Aggregation (Recommended)

Teacher collects data and submits only class averages:
- Mean yawns per day
- Distribution by time of day
- Distribution by activity
- Age range and number of participants

### Option 3: No Submission

Use entirely as a learning activity without submitting data to the project.

## Troubleshooting Common Issues üîß

### "I keep forgetting to track!"

**Solutions**:
- Set phone reminders
- Use stickers on hand
- Partner buddy system
- Track at specific times only (approximate)

### "I don't yawn much"

**Response**: That's interesting data! Some people yawn less frequently, and that's important to know.

### "This is boring"

**Solutions**:
- Add competitive element (who can predict their count best?)
- Focus on interesting patterns they notice
- Connect to real-world applications
- Show surprising yawn facts

### "Students are fake-yawning"

**Response**: Discuss scientific integrity. Explain that fake data helps no one and defeats the learning purpose.
