# Quick Start Guide

## Steps to Start

### 1. Choose Your Method âœ“

Pick the tracking method that works for you:

**Option A: Simple Paper Tracking**
- Print the [simple tracking sheet](../data/templates/yawn-tracking-simple.csv)
- Keep it with you
- Make a tally mark for each yawn
- Note the time and what you're doing

**Option B: Digital Spreadsheet**
- Download [yawn-tracking-simple.csv](../data/templates/yawn-tracking-simple.csv)
- Open in Excel, Google Sheets, or similar
- Add a row for each yawn
- Save regularly

**Option C: Phone Notes**
- Use your phone's notes app
- Create a simple log format
- Type entries as you yawn
- Transfer to template later

### 2. Create Your Anonymous ID ğŸ”

**Important**: Never use your real name!

**Create an ID like this:**
```
YI-[year]-[initials]-[random numbers]
Example: YI-1990-JD-742
```

Or just make up a random ID:
```
YI-ABC-123-XYZ
```

Write it down somewhere safe - you'll need it for all your entries!

### 3. Start Tracking ğŸ¯

**For each yawn, record:**

âœ… **Date**: Today's date (YYYY-MM-DD format)
âœ… **Time**: When it happened (24-hour format like 14:30)
âœ… **Duration**: Short, medium, or long yawn
âœ… **Context**: What you were doing
âœ… **Environment**: Indoor or outdoor

**Example entry:**
```
YI-1990-JD-742,2025-01-15,14:30,medium,desk-work,indoor
```

### 4. Track for 7 Days ğŸ“…

- Try to catch every yawn (or as many as you can!)
- Don't worry if you miss some
- Be consistent with your format
- Keep your tracking sheet handy

**Pro tip**: Set a reminder on your phone to review your tracking at the end of each day.

### 5. Complete Daily Summaries ğŸ“Š

At the end of each day, note:

- **Total yawns** for the day
- **Sleep quality** last night (poor/fair/good/excellent)
- **How you felt** (energy level, stress level)
- **Any exercise** (minutes)

**Example:**
```
YI-1990-JD-742,2025-01-15,12 yawns,7.5 hours sleep,good quality,low stress,30 min exercise
```

## Tips for Success ğŸ’¡

### Tracking Tips
- âœ… Keep your tracking sheet accessible
- âœ… Update in real-time when possible
- âœ… Set phone reminders if needed
- âœ… Don't worry about perfection
- âœ… Notes can be very brief

### Common Contexts
Use these standard categories:
- `waking-up` - Just after waking
- `morning-routine` - Morning activities
- `desk-work` - Computer/office work
- `eating` - During meals
- `physical-activity` - Exercise
- `watching-media` - TV, videos
- `relaxing` - Leisure time
- `before-sleep` - Evening routine
- `other` - Anything else

### Duration Guide
- **Short**: Quick yawn (<3 seconds)
- **Medium**: Normal yawn (3-5 seconds)
- **Long**: Big, extended yawn (>5 seconds)
