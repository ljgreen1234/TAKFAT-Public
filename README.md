# TAKFAT-Yawn-Index

## A Citizen-Science Experiment on Yawning and Autonomic Health

Welcome to the TAKFAT-Yawn-Index project! This is an open, community-driven research initiative exploring the fascinating connection between yawning patterns and autonomic nervous system function.

### ğŸ¯ Project Hypothesis

**Can yawn persistence (Yawn Index) reveal patterns tied to autonomic health, including long-term COVID effects?**

Yawning is a universal behavior that may be linked to autonomic nervous system regulation. By tracking yawning patterns alongside optional physiological data (heart rate variability, gait patterns), we aim to discover whether there are meaningful correlations between yawn frequency and autonomic health markers.

- **Educational**: Learn about the autonomic nervous system in a fun, engaging way
